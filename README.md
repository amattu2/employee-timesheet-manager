# Description
A Python 3.6 based employee timesheet manager. Implements a localized database and PDF export option.

# Dependencies
- Tkinter (Python3)
- FPDF (Included, https://github.com/reingart/pyfpdf)
- TinyDB (Included, https://github.com/msiemens/tinydb)
